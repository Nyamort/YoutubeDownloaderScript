# YoutubeDownloaderScript

## Install
With NPM
```
npm install
```
With yarn
```
yarn install
```

## Use
```
node index.js youtube_link start_time end_time
```
example :
```
node index.js https://www.youtube.com/watch?v=D1xBiK4UKcA&t 00:00:05 00:01:53 
```
## Description
Download and cut your youtube video
